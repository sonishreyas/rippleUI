Avatars

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Text Avatar

We can use text initials for text avatar. You will need to include the avatar, avatar-text and the size class. These are available in four sizes xl, l, m, and s.

Profile Avatar

We can use user profile picture for avatar. You will need to include the avatar,and the size class. These are available in four sizes xl, l, m, and s.

Status Avatar

We can use user status avatar for getting the status of the user. You will need to include the avatar, avatar-badge, avatar-badge-xl (this will change based on sizes), and the size class. These are available in four sizes xl, l, m, and s.
